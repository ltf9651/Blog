## 01 | 复杂度分析

1. 只关注循环执行次数最多的一段代码
    ```C
    int cal(int n) {
    int sum = 0; // 1 unit_time
    int i = 1;   // 1 unit_time
    for (; i <= n; ++i) {
        sum = sum + i;  // n unit_time
    }
    return sum;
    }

    O(n) = 1 + 1 + n => n
    ```

1. 加法法则：总复杂度等于量级最大的那段代码的复杂度
    ```C
    int cal(int n) {
    int sum_1 = 0;
    int p = 1;
    for (; p < 100; ++p) {
        sum_1 = sum_1 + p; // 100 => 1 unit_time
    }
    
    int sum_2 = 0;
    int q = 1;
    for (; q < n; ++q) {
        sum_2 = sum_2 + q; // n unit_time
    }
    
    int sum_3 = 0;
    int i = 1;
    int j = 1;
    for (; i <= n; ++i) {
        j = 1; 
        for (; j <= n; ++j) {
        sum_3 = sum_3 +  i * j; // n2 unit_time
        }
    }
    
    return sum_1 + sum_2 + sum_3;
    }

    O(n) = 1 + n + n^2 = n^2
    ```

1. 乘法法则：嵌套代码的复杂度等于嵌套内外代码复杂度的乘积
    ```C
    int cal(int n) {
    int ret = 0; 
    int i = 1;
    for (; i < n; ++i) {
        ret = ret + f(i);// n^n unit_tine
    } 
    } 
    
    int f(int n) {
    int sum = 0;
    int i = 1;
    for (; i < n; ++i) {
        sum = sum + i;   // n unit_tine
    } 
    return sum;
    }

    O(n) = n^2
    ```
